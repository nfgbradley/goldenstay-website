## 📌 Description

GOLDENSTAY est une plateforme moderne permettant de :
- Réserver des appartements de luxe
- Louer des véhicules premium
- Contacter directement via WhatsApp
- Consulter les disponibilités en temps réel

---

## 🧱 Structure du projet

### 📄 index.html
Contient :
- Header (logo + navigation)
- Hero section (bannière principale)
- Sections appartements & véhicules
- Bouton WhatsApp flottant
- Système de réservation

---

### 🎨 style.css
Design premium :
- Thème noir + accents dorés
- Layout responsive (mobile / desktop)
- Cartes modernes type Airbnb / Booking.com
- Hover effects et animations

---

### ⚙️ script.js
Fonctionnalités :
- Génération de liens WhatsApp
- Création dynamique des véhicules (21 voitures)
- Système de réservation
- Bouton WhatsApp flottant

---

## 🚀 Installation locale

1. Télécharger le projet
2. Ouvrir `index.html` dans un navigateur
3. Ou utiliser Live Server (VS Code recommandé)

---

## 🌐 Déploiement GitHub Pages

1. Créer un repository sur GitHub
2. Uploader tous les fichiers :
   - index.html
   - style.css
   - script.js
   - dossier /public (images)
3. Aller dans **Settings → Pages**
4. Activer :
5. Le site sera disponible ici : https://username.github.io/repository-name
