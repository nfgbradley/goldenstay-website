# GOLDENSTAY 🏨🚗

Premium website for booking luxury apartments and vehicles in Brazzaville.

## ✨ Features
- 🏢 Luxury apartments showcase (Luxe 1, 2, 3)
- 🚗 Vehicle rental (Toyota Prado fleet)
- 📲 Direct WhatsApp reservation system
- 🎬 Automatic background slider (hotel-style)
- 🎨 Modern premium UI (black & gold theme)
- 📱 Fully responsive design
