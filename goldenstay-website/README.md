# GOLDENSTAY Website

Welcome to the GOLDENSTAY website project! This project is designed to showcase premium apartments and vehicles available for rent. Below are the details of the project structure and functionality.

## Project Structure

- **index.html**: The main HTML file that contains the structure of the website, including:
  - A header with the logo and navigation menu.
  - A hero section featuring a background image and overlay.
  - Sections for displaying apartments and vehicles in a responsive grid layout.
  - A footer with contact information.
  - A floating WhatsApp button for easy reservations.

- **style.css**: The CSS file that defines the premium design of the website, including:
  - Background color set to black and text color to white.
  - Golden accents for a luxurious feel.
  - Styles for the hero section, overlay, responsive grid layout, and hover effects for cards.

- **script.js**: The JavaScript file that adds functionality to the website, including:
  - A function to generate WhatsApp reservation links.
  - Dynamic generation of vehicle cards for 21 different cars.
  - Implementation of a floating WhatsApp button for user convenience.

## Getting Started

To get started with the GOLDENSTAY website, simply open the `index.html` file in your web browser. You can customize the content and styles as needed to fit your branding and offerings.

## Features

- Responsive design that adapts to various screen sizes.
- Premium look and feel with a focus on user experience.
- Easy access to reservations via WhatsApp.
